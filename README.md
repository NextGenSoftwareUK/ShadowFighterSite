# Shadow Fighter — The Trilogy

**OASIS Omniverse · Fighting Game Trilogy**

Shadow Fighter is a fighting game trilogy integrated with the OASIS Omniverse. Crimson/electric blue theme, classic 2D fighter gameplay, and a deep roster of fighters whose special moves and unlocks are tied to real-world karma and NFT ownership.

## The Trilogy

| Game | Description |
|---|---|
| **Shadow Fighter I** | The original — 8 fighters, 6 stages, classic arcade mode |
| **Shadow Fighter II** | Extended roster, online multiplayer, tournament mode |
| **Shadow Fighter III** | Full story mode, 3D arenas, cross-game NFT fighter skins |

## OASIS Integration

- **Avatar SSO** — your OASIS avatar links to a fighter profile
- **Karma unlocks** — reaching karma milestones unlocks hidden fighters, stages and cosmetics
- **NFT fighters** — rare fighter skins are on-chain NFTs; own them across all devices
- **Tournament karma** — winning OASIS tournaments earns large karma bonuses
- **Cross-game items** — weapons collected in OQUAKE and ODOOM can appear as in-game power-ups
